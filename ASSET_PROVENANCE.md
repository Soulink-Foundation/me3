# Asset Provenance

This file tracks public assets and sample content that ship with ME3 Core.

## Included Assets

- `apps/web/public/favicon.ico`, `apps/web/public/favicon.png`, `apps/web/public/icons/apple-touch-icon.png`, `apps/web/public/icons/icon-192.png`, `apps/web/public/icons/icon-512.png`: ME3 app icons derived from the project brand mark. They are included for running and forking the template.
- `apps/web/public/me3-logo-dark.png`, `apps/web/public/me3-logo-light.png`: ME3 project logo files. The MIT License covers repository redistribution, but the ME3 name and logos should not be used to imply endorsement by the original project maintainers.
- `apps/web/public/portable-index.html`: generated portable site template used by the site export flow. It contains neutral template code and no personal demo content.

## Removed During Public-Readiness Audit

- `apps/web/public/me3protocol.jpg`: copied landing/demo image with unclear provenance.
- `apps/web/public/robot.png`: copied assistant illustration with unclear provenance and hosted-app branding assumptions.
- `apps/web/public/flower.png`: unused copied decorative asset with unclear provenance.

## Sample Content Rules

Fixtures, placeholders, and demo text should use neutral example names, `example.com` domains, and synthetic profile data. Do not add real people, customer content, hosted production URLs, or third-party brand assets unless their license and provenance are recorded here.
